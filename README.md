# 🚑 RapidAid – Emergency Assistance App

RapidAid is a mobile application designed to provide **quick and reliable emergency assistance**. The goal of this project is to help users get immediate support in critical situations like medical emergencies, accidents, or urgent help requests.

---

## 📌 Features (Initial Version)

* 📍 **Location Tracking** – Get user's current location for emergency response
* 🚨 **Emergency Request** – Send help requests instantly
* 📞 **Quick Contact** – Connect with emergency contacts/services
* 🧭 **Simple UI** – Easy to use interface for fast interaction

---

## 🛠️ Tech Stack

* **Frontend:** Android (Jetpack Compose / XML)
* **Architecture:** MVVM (Model-View-ViewModel)
* **Backend (Planned):** Firebase / Custom API
* **Language:** Kotlin
* **Database (Planned):** Firebase / SQLite

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/rapidaid.git
```

### 2. Open in Android Studio

* Open Android Studio
* Click on **Open Project**
* Select the RapidAid folder

### 3. Run the App

* Connect an emulator or physical device
* Click **Run ▶️**
